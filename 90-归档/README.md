# 归档

这里存放废弃、过期、不再维护或只保留历史上下文的内容。

## 使用规则

- 归档不是删除。归档材料用于保留历史判断和失败背景。
- Agent 不能把归档内容当作当前结论。
- 如果归档内容仍有参考价值，应在正文开头说明它为什么被归档、还能参考什么。

## 建议 frontmatter

```yaml
---
title: 归档材料标题
status: deprecated
type: note
domain: ops
maintainer: Billy
updated: 2026-06-07
tags:
  - archived
agent_reading: ignore
---
```
