# Skills 索引

这里记录团队关心的 skills。Skills 按使用场景分成开发和科研两类。

## 分类

- [[开发Skills]]
- [[科研Skills]]

## 使用原则

- 先判断任务属于开发、科研还是混合任务。
- 开发任务优先看 [[开发Skills]]。
- 科研任务优先看 [[科研Skills]]。
- 如果一个 skill 只是临时试用，先标记为 `draft`，不要直接当作团队标准。
