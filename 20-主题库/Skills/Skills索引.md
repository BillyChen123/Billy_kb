# Skills 索引

这里记录团队关心的 skills。后续每个稳定 skill 可以单独成文，说明用途、触发方式、适用边界和维护经验。

## 通用

### Superpower

- 用途：头脑风暴。
- 适合场景：从粗糙想法推到结构化设计；先讨论再实现。
- 后续沉淀：团队如何用它设计项目、方法范式和知识库结构。

## 开发

### ui-ux-pro-max + design.md

- 用途：前端设计。
- 适合场景：网页、应用、仪表盘、组件等 UI/UX 设计与实现。
- 后续沉淀：前端项目的设计输入模板和验收清单。

### remotion

- 用途：视频制作。
- 适合场景：用 React/Remotion 生成视频内容。
- 后续沉淀：视频项目模板、素材组织方式、渲染排错经验。

## 科研

### nature-skills

- 用途：科研绘图。
- 适合场景：论文图、模型图、机制图、科研展示图。
- 后续沉淀：高质量科研图的 prompt 和审美标准。

### PPT-master

- 用途：组会、汇报、演示文稿。
- 适合场景：组会 PPT、课题汇报、论文解读。
- 后续沉淀：组会汇报模板和科研叙事结构。
