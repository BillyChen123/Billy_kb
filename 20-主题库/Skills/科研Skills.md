# 科研 Skills

这里放面向科研写作、科研绘图、文献整理、组会汇报和高水平论文表达的 skills。

## nature-skills

GitHub：

- https://github.com/Yuan1z0825/nature-skills

它有什么用：

- 面向科研人员的 Nature 风格学术表达和科研绘图 skill 集合。
- 目标是让论文写作、图表表达、引用、数据可用性、审稿回复等环节更接近 Nature / Science / Cell 等高水平期刊标准。
- 适合把通用“润色”和“画图”提升成面向高水平论文的结构化工作流。

适合场景：

- 论文图、机制图、模型图、multi-panel figure。
- Nature 风格论文语言润色。
- 学术引用、数据可用性声明、审稿回复。
- 高水平期刊投稿前的表达质量检查。

在本库中的定位：

- 科研绘图和高水平论文表达 skill。
- 后续可以沉淀团队自己的科研图审美标准、prompt、figure checklist。

## PPT-master

GitHub：

- https://github.com/hugohe3/ppt-master

它有什么用：

- AI 驱动的 PPT 生成系统，可以从 PDF、网页、Markdown 或文本材料生成高质量演示文稿。
- 输出目标是可编辑的真实 PowerPoint，而不是不可编辑的图片。
- 工作流包含源内容处理、项目初始化、设计规范、SVG 页面生成、演讲备注、后处理和 PPTX 导出。

适合场景：

- 组会 PPT。
- 课题汇报。
- 论文解读。
- 项目阶段性总结。
- 把已有 Markdown 或论文材料转成可展示的演示文稿。

在本库中的定位：

- 科研汇报和组会 skill。
- 后续可以沉淀组会汇报模板、科研叙事结构、PPT 质量验收清单。
