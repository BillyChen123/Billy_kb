# 项目

这里按课题或项目聚合材料。项目是最重要的人类入口，也是 Agent 同步上下文的主要单位。

## 项目结构

```text
项目名/
  README.md
  memory.md
  notes/
  outputs/
  lessons/
```

## 放什么

- 正在推进的课题
- 已完成但有复用价值的课题
- 需要多人或多轮 Agent 协作的项目
- 有讨论、实验、输出、经验沉淀的工作

## 不放什么

- 跨项目通用知识，放 `20-主题库/`
- 文档模板，放 `30-模板/`
- 临时未分类材料，放 `01-收件箱/`

## Agent 规则

处理项目时，必须先读：

1. `README.md`
2. `memory.md`

每次实质推进后，判断是否需要 `/sync_memory`。
