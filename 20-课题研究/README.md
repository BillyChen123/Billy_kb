# 课题研究

这里存放课题相关材料。这个目录的目标不是把所有东西格式化成论文，而是沉淀已经探索过的路径、走得通的经验、走不通的路线，以及正在推进的合作讨论。

## 子目录

```text
20-课题研究/
  00-课题地图/
  10-前沿探索/
  20-讨论合作/
  30-经验沉淀/
  40-对标论文/
  90-归档/
```

## 放置规则

- `00-课题地图/`：课题列表、方向索引、问题地图。
- `10-前沿探索/`：前沿方向、通用算法、创新点、paper 线索。
- `20-讨论合作/`：正在推进的课题讨论、阶段进展、会议记录。
- `30-经验沉淀/`：已经做过的课题经验、实验路径、失败路线、复盘。
- `40-对标论文/`：高水平期刊论文、目标论文、对比对象。
- `90-归档/`：过期课题材料和不再维护的研究记录。

## 课题材料的核心价值

研究区最重要的不是“写得漂亮”，而是保留判断过程：

- 为什么当时选择这个方向？
- 试过哪些方法？
- 哪些方法有效？
- 哪些路线浪费时间？
- 还有哪些待验证问题？

失败路线要保留。它们是后续 Agent 和成员避免重复踩坑的关键材料。
