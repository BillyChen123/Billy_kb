# 如何让 Agent 读取知识库

Agent 的主规则在根目录 `AGENTS.md`。本文件只保留最短操作说明。

## 必读顺序

1. 先读 `AGENTS.md`。
2. 如果处理项目，读 `10-项目/项目名/README.md`。
3. 再读 `10-项目/项目名/memory.md`。
4. 需要背景时再查 `notes/`、`outputs/`、`lessons/` 和 `20-主题库/`。

## 项目任务默认动作

每次完成实质推进后，Agent 必须判断：

- 当前目标有没有变化？
- 当前状态有没有变化？
- 是否产生新的关键判断？
- 是否发现不要重复踩的坑？
- 下一步是否需要更新？
- 是否有关键材料需要链接？

如果答案是 yes，就执行 `/sync_memory`，更新项目 `memory.md`。

## 常用操作协议

- `/sync_memory`：同步项目当前记忆。
- `/capture_lesson`：沉淀本轮经验或失败路线。
- `/promote_knowledge`：把收件箱或过程材料整理成正式知识。
- `/start_project`：创建新项目结构。

这些可以是真命令，也可以是用户对 Agent 的自然语言指令。
