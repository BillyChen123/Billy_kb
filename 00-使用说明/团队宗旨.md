# 团队宗旨

这是 Billy_kb 的协作底色。人和 Agent 都应该读到。

## 核心宗旨

- 苟富贵，勿相忘。
- 忠诚！忠诚！忠诚！

## 对协作的含义

- 这个知识库服务长期协作，不服务一次性消耗。
- 有价值的经验要沉淀下来，作为下一轮 Agent、下一个项目、下一个人的复利。
- 过程可以粗糙，但关键判断、踩坑经验和下一步必须尽量同步到项目 `memory.md`。
