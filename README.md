# Billy_kb

这是团队协作知识库的 Obsidian vault。

## 给成员的最短说明

1. 用 Obsidian 打开这个文件夹。
2. 平时主要在 `01-投稿箱/你的名字/` 写内容。
3. 不要修改 `AGENTS.md`。
4. 不要把密码、token、公司隐私信息写进这个仓库。
5. 遇到同步冲突，先不要覆盖，找维护者处理。

## 目录

- `00-使用说明/`：使用说明、接入说明
- `01-投稿箱/`：成员投稿区，默认视为未审内容
- `10-AI-Agent知识/`：正式知识区
- `20-课题研究/`：正式研究资料
- `30-Skills-MCP-工具资源/`：工具、技能、MCP 资料
- `90-归档/`：归档和废弃内容
