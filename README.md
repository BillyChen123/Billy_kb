# Billy_kb

这是团队协作知识库的 Obsidian vault。

## 这是什么

这个仓库同时服务两类协作者：

- 人类成员：主要用 Obsidian 写作、浏览、整理材料
- Agent：读取和维护正式知识区

仓库当前采用单仓库协作方式，所有成员共用同一个 vault。

## 给成员的最短说明

1. 先看详细接入文档：[00-使用说明/如何接入Obsidian.md](./00-%E4%BD%BF%E7%94%A8%E8%AF%B4%E6%98%8E/%E5%A6%82%E4%BD%95%E6%8E%A5%E5%85%A5Obsidian.md)
2. 日常主要在 `01-投稿箱/你的名字/` 下写内容。
3. 不要修改 `AGENTS.md`。
4. 不要把密码、token、公司隐私信息写进这个仓库。
5. 遇到同步冲突，先不要覆盖，先找维护者处理。

## 目录规则

- `00-使用说明/`：使用说明、接入说明、协作规范
- `01-投稿箱/`：成员投稿区，默认视为未审内容
- `10-AI-Agent知识/`：正式知识区
- `20-课题研究/`：正式研究资料
- `30-Skills-MCP-工具资源/`：工具、技能、MCP 资料
- `90-归档/`：归档、废弃、过期内容

## 协作边界

- 投稿内容优先放 `01-投稿箱/`
- 正式知识整理再进入 `10/20/30` 目录
- `01-投稿箱/` 中的内容默认不是正式事实来源

更详细的 Agent 规则见 [AGENTS.md](./AGENTS.md)。
